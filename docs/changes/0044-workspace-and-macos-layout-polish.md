# 0044：工作区、初始向导与 macOS 窗口细节

日期：2026-07-22

状态：Windows 已通过实机验收（2026-07-22）；macOS 待实机执行。

## 变更

- 文件夹选中态使用当前主题色的淡色背景和文字；邮件列表自绘滚动槽在可滚动时始终可见。
- 账户身份不再显示“正在同步”，离线、重试和重新认证等需要用户感知的状态仍保留。
- 初始向导右栏改用不占布局宽度的自绘滚动槽，并让向导背景延伸到覆盖式标题栏下，移除重复的标题栏空白带。
- 两处分栏 grid 轨道从 1 px 改为 0，宽拖拽命中区继续覆盖相邻容器内部，消除三栏间物理空隙。
- 主窗口、设置窗口和写信窗口删除 macOS 交通灯硬编码坐标，交由系统在 Overlay 标题栏中定位。

## 验收限制

- 自动测试和 Windows 构建不能证明 macOS 交通灯位置正确；该项必须在 macOS 12+ 主/设置/写信三个窗口分别检查。
