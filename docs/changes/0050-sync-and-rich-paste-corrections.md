## 变更

- Supervisor 完整同步只保留账户首次设定/应用启动、按账户的手动/1/5/10 分钟间隔和手动收取；删除 IDLE、轮询兜底、秒级失败重试及设置/待办误触发完整同步的路径。
- 数据格式版本 19 为每个匿名账户槽持久化同步间隔，默认 1 分钟；修改正文策略只保存，修改间隔只重置计时。
- 邮件逐封原子落库后继续发送最小事件；当前文件夹按事件串行重新读取本地 Query，删除前端 40ms 模拟呈现队列，并保留已有本地首屏立即显示和 IMAP 100 UID 批取性能。
- `account-runtime-status-changed` 纳入主工作区 Query 失效；任何真实同步都会禁用手动收信，Rust 同时拒绝重复同步竞态。
- 设置侧栏选中项统一使用 `bg-primary/10` 与 `text-primary`。
- Composer 工具栏新增图片选择；选择、图片粘贴和富 HTML 剪贴板图片复用现有 CID 内容寻址缓存。
- 新增窄 `sanitize_rich_text_paste` Command；安全 HTML 以独立作用域保留 class/ID、行内样式和样式表，Tiptap 保留对应可编辑结构。

## 边界
