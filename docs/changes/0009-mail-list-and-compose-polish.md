# 变更 0009：邮件列表与写信入口精修

日期：2026-07-12

## 变更内容

- 邮件列表取消卡片间距与圆角，每封邮件连续排列，并使用底部分隔线建立行边界；选中状态继续使用背景层级表达。
- “新建邮件”从全局工具栏移动到左侧文件夹区域顶部，使用强调色主按钮。
- 本地草稿入口与“新建邮件”组成连体 split button；没有本地草稿时不显示下拉段。
- 草稿菜单内容改为自适应高度，增加菜单外边距、项目上下内边距和稳定行高，避免标题/收件人贴近菜单边缘。
- 移除 Tiptap 编辑区域 focus 时的左侧强调线，保留原生光标与工具栏状态作为焦点反馈。
