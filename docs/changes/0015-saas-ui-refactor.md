# 变更 0015：跨平台窗口壳与 SaaS UI 重构

日期：2026-07-13

- 新增 Windows 无系统装饰的自绘标题栏，以及 macOS Overlay 标题栏与原生交通灯平台配置。
- 主、写信、设置窗口补充最小化、最大化、关闭和拖动权限；设置窗口使用独立 Capability。
- 新增单例设置窗口及分类侧栏，接入语言、主题、强调色、账户同步策略、文件夹映射与关于信息。
- 偏好持久化后发布 `appearance-preferences-changed`，各 WebView 即时应用主题和语言。
- 引入随包 Roboto 与 Droid Sans Fallback，更新浅色/深色、表面、强调色、选中态、圆角、阴影和焦点令牌。
- 基础按钮、输入框、选择器、弹层、卡片、提示、Toast 和写信/首次启动界面统一为无装饰边框样式。
- 移除主顶部工具栏；账户菜单接管手动收取、设置和退出。
- 重构沉浸式侧栏、文件夹主操作、邮件列表标题/统计/搜索/选中态，以及阅读器图标操作区。
- 保留现有同步、草稿、SMTP、回复/回复全部/转发、移动、复制、删除和归档业务命令，不改变数据或协议边界。
