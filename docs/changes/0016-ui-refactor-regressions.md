# 变更 0016：UI 重构回归修复

日期：2026-07-13

- 折叠文件夹栏的新建邮件按钮改为固定方形并在可用宽度中严格居中。
- 所有折叠文件夹图标使用固定 `18 × 18` 尺寸和禁止收缩，避免窄栏中 SVG 比例异常。
- 移除两个栏宽拖拽命中区默认显示的 1px 竖线；拖拽和键盘调整能力保持不变。
- 设置窗口改用账户摘要查询，不再为读取账户列表加载完整首次启动状态。
- 设置 Capability 增加仅限该窗口的 destroy 权限；关闭按钮直接销毁设置 WebView，避免失效窗口驻留。
- 窗口内容增加生产级错误边界。设置动态模块或页面渲染异常时仍保留自绘标题栏，并提供可用的关闭入口，不再留下无法操作的纯白窗口。
- 增加折叠图标尺寸/居中与独立设置窗口首屏渲染回归测试。
