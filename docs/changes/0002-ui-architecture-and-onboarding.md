# 0002 - UI 架构与首次启动体验调整

日期：2026-07-12

## 功能变更

- 首次设置增加正式欢迎页，形成欢迎、数据目录、第一个账户三个步骤。
- 主界面和首次设置侧栏移除主题与强调色入口，仅保留语言切换；外观能力保留给后续偏好设置页面。
- 修复账户输入框因在 React 状态更新器中延迟读取事件对象而触发的空引用错误。
- 语言偏好统一使用 `zh-CN` 和 `en-US`，Rust 同时兼容读取旧的 `zh-cn` 和 `en-us`。

## 前端架构变更

- 将原 `ui/index.tsx` 拆为 `src/components/ui/` 下的单组件文件。
- 组件采用 shadcn 的源码归属和组合方式，Radix Primitives 提供可访问交互，视觉不使用预设默认样式。
- 引入 `components.json` 和 `@/*` 路径别名，采用 Nova 紧凑密度与 Lyra 利落几何的自定义主题。
- Tailwind v4 用于组件和页面样式；CSS 拆分为主题令牌与全局基础样式，不再使用单一大型 `global.css`。

## 文档变更

- 新增文档索引和总体实施计划。
- 记录 Cargo Workspace 的渐进式拆包目标；当前阶段不提前创建空子包。
