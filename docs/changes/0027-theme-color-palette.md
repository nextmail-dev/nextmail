# 0027 设置页主题色圆点色板

日期：2026-07-18

## 变更

- 设置 → 外观中的“强调色”改为更易理解的“主题色”。
- 移除主题色下拉框，改用可直接预览颜色的圆点色板。
- 色板从蓝色、紫罗兰、青色和玫红扩展为蓝色、靛蓝、紫罗兰、紫色、玫红、红色、橙色、琥珀色、绿色和青色。
- 圆点使用原生单选语义，支持 Tab、方向键和空格键操作，并提供中英文可访问名称。
- 内部配置字段继续使用 `accentColor`，已有四种颜色的值保持不变；若读到旧版本保存的其他合法颜色，设置页会额外显示该颜色，不触发数据迁移或偏好重置。

## 验证

- 设置页组件测试覆盖色板数量、当前选中状态和颜色切换持久化调用。
- `pnpm exec tsc --noEmit` 通过。
- `pnpm exec vitest run` 通过，共 13 个测试文件、28 项测试。

## 手动验收

- 2026-07-18：主题色名称、圆点色板、扩展颜色和即时切换通过用户实机验收。
