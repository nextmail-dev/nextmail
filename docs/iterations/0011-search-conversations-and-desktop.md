# 第十一阶段：搜索、会话与桌面集成

状态：实施中。第一批“当前文件夹本地全文搜索”已于 2026-07-22 通过 Windows 手动验收；后续批次尚未实施。

## 阶段定位

在模板/签名和邮件阅读/回复体验稳定后，为本地邮件数据依次补齐全文检索、会话聚合和桌面常驻能力。本阶段按可独立验收的功能批次推进，不把搜索、会话、托盘、窗口和通知一次性交付。

## 已确认边界

- 第一批沿用现有“搜索当前文件夹”入口，只搜索当前公开账户、当前文件夹和本地已同步数据；不引入跨账户搜索或统一收件箱。
- FTS 语料为主题、发件人/收件人/抄送地址、预览、正文纯文本和附件名。HTML 标记、原始 EML、密码、Token、内部路径和服务器错误不进入索引。
- 搜索文本按字面连续内容处理，不把用户输入解释为 FTS 操作符。三个及以上 Unicode 字符使用 FTS5 trigram；一至两个字符使用同一索引内容的受限短词扫描，以支持常见中文短词。
- 搜索结果保持现有邮件列表 DTO、日期游标分页和排序。事件仍只使账户/文件夹 Query key 失效，不推送结果或正文。
- 统一收件箱仍是总体计划明确的非目标。会话、托盘和通知批次不会借机加入联系人、规则、OAuth、POP3、自动更新或全局优化。

## 实施批次

### 第一批：当前文件夹本地全文搜索

状态：已于 2026-07-22 通过 Windows 手动验收。

- 数据格式版本 15 新增 `message_search` FTS5 表；迁移会为既有邮件回填索引。
- 数据库触发器随邮件头/预览、纯文本正文和附件名的插入、更新、删除维护索引；同步仍在既有事务内提交可见数据。
- Rust 提供独立薄 Command `search_messages`，先把公开账户 ID 解析为匿名 `account_slot_id`，Repository 查询再同时约束账户槽、文件夹与可见位置。
- 前端通过 `src/app/api.ts` 调用，250ms 防抖后使用 `['messages', accountId, mailboxId, 'search', query]` Query key；清空搜索恢复普通分页列表。
- 搜索结果可能由正文、收件人或附件名命中，因此前端不再对列表 DTO 做第二次内存过滤。

自动验证：

- Rust 回归覆盖主题、地址、正文、附件名、中文短词、分页、索引更新、文件夹隔离和账户槽隔离。
- 前端回归覆盖防抖、稳定 Query key 对应的服务端搜索，以及正文/附件命中结果不被列表字段过滤。
- 完整命令门禁在本批交付前执行并记录到 `docs/changes/0040-local-message-search.md`。

Windows 手动验收：

1. 使用 `pnpm tauri dev` 启动已有数据目录，确认迁移后原有账户、文件夹和邮件仍可正常打开。
2. 在某个邮件较多的文件夹中，分别搜索主题片段、发件人姓名/邮箱、收件人、正文中存在但列表摘要没有的词，以及附件文件名；结果应覆盖尚未手动“加载更多”的本地邮件。
3. 搜索一个一至两个汉字的词，再搜索一个三字以上中文片段；两者都应返回本地匹配项。
4. 在两个文件夹放置不同的可匹配邮件，确认当前搜索不返回另一文件夹结果；有多个账户时确认不会出现其他账户邮件。
5. 在结果多于 50 条的文件夹滚动并点击“加载更多”，确认没有重复、跳页或混入非匹配邮件。
6. 清空搜索，确认恢复当前文件夹普通列表；切换文件夹或账户后搜索框清空，邮件阅读、已读和星标仍正常。
7. 收取一封包含独特主题/正文/附件名的新邮件后直接搜索，确认无需重启即可命中。

验收结果：上述 Windows 实机流程通过。既有数据迁移、各类本地语料搜索、中文短词、分页、账户/文件夹隔离、新邮件入索引及普通列表恢复均确认正常；macOS 尚未执行。

## 安全与数据边界

- FTS 数据位于可迁移 `content.sqlite`，属于可重建本地视图；所有搜索查询必须同时按 `account_slot_id` 和 `mailbox_id` 隔离。
- 系统通知不得包含密码、Token、内部路径、服务器原始错误或未经确认展示的敏感正文。
- 托盘和通知只通过窄 Tauri Command/Event 与平台 Capability 接入，不向 React 开放通用系统 API。
- 不持有 SQLite 写锁等待网络、慢文件 I/O 或系统通知调用。

## 非目标

- 不实现跨账户搜索、统一收件箱、远端服务器搜索、联系人、规则、日历、PGP/S-MIME、企业策略或遥测。
- POP3、OAuth、发布硬化和全局优化仍为未排期设想。
- 本阶段不运行 Tauri bundle，不创建发布、签名或自动更新配置。
